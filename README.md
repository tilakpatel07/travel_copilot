## Travel Copilot

Generative AI Itinerary Planning for Gen Z Travelers

One-Line Summary

Travel Copilot is a Generative AI product that creates personalized, budget-aware 3-day itineraries for Gen Z travelers, designed with strict latency, accuracy, and trust constraints.

### Problem

Gen Z travelers plan 3–5 trips per year but face:

- Information overload across fragmented sources (TikTok, blogs, booking sites)

- Low trust in generic AI outputs due to hallucinations and poor logistics

- High expectations for speed and personalization

- Existing tools optimize for search or booking—not decision quality.

### Product Solution

A mobile-first AI Travel Copilot that:

- Instantly generates personalized 3-day itineraries based on interests and budget

- Maintains low latency (≤1.5s) for seamless UX

- Uses guardrails + feedback loops to minimize hallucinations and build trust

- Balances AI automation with user control

### Target User

- Gen Z (18–27)

- Digital natives

- Budget-aware

- Increasingly open to AI—but only if it’s reliable
